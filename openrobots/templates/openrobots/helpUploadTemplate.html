Description template fields


Helpful hints to fill the form for adding new template protocol file
  • Upload Template file:

    It is the python file with the actions to run the protocol on the Robot


  • Protocol description to be displayed in Form:

    Write the name that will display in the form when investigator select the protocol template



  • Select Protocol Project:

    Select from the option list the project from where the protocol template file belongs to


  • Select Station and Protocol

    To help you to identify the right station and the protocol, in the selection option it is showed in this format:
    "Name of the station" -- "main use for the station" -- Protocol Number


  • Protocol Template Version

    Type the version assigned for this protocol template
